## 项目简介

本项目是面向智慧水利的洪水预警与决策支持系统，通过集成实时雨情监测、深度学习预测模型、GIS可视化和应急决策支持，为流域防洪提供全方位的智能化解决方案。

### 核心功能

⛈️ **实时雨情监测**：接入气象数据API，实时监测降雨情况  
🦕 **洪水预测**：基于LSTM深度学习模型，预测未来洪水过程  
🗺️ **淹没模拟**：GIS可视化展示淹没范围和影响区域  
🎯 **决策支持**：提供人员撤离方案、资源配置建议  
📊 **数据可视化**：ECharts交互式图表展示
